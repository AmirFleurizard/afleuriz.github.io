Project Design

Project Overview

The purpose of this application is to serve as a portfolio for my client. The intended users of this client will exclusively be my client and potential employers the client decides to share the portfolio with. The client has laid out that they would like the website to include a bio that she will provide, work history, pictures of what she has accomplished in her career, and her resume.

Page Design

Name of Page Purpose of Page Users of Page Content of the Page Will this page ask for users to enter data? Data Field Validations Does this page contain buttons? What actions will happen on this page?
Home The purpose of this page is to serve as an entry way into the website, and provide a brief overview of the site. This page is for potential employers. The content on the page will be an overview of the website and its purpose. No, this page is not a place for users to enter data. n/a The page will contain a hyperlinks to the other pages When the user clicks a button it will navigate them to the other pages of the site based on which button they click.
About Me The purpose of this section is to tell the user about the client in hopes of them connecting on a more personal level. The audience for this page would be potential employers. The content on the page will be a description of the client and what they are passionate about This is not a page where a user can enter data. n/a The page will contain hyperlinks to the other pages. When the user clicks a button it will navigate them to the other pages of the site based on which button they click.
Resume The purpose of this page is to show the clients resume. The audience for this page will be potential employers The content for this page will just be the clients resume. This is not a page where the users will enter data. The page will not need data validations. The page will contain hyperlinks to the other pages. When the user clicks a button it will navigate them to the other pages of the site based on which button they click.
Work The purpose of this page is to show the clients relevant work. The audience for this page will be potential employers The content for this page will include images of the clients relevant work. This is not a page where the users will enter data. The page will not need data validations. The page will contain hyperlinks to the other pages. When the user clicks a button it will navigate them to the other pages of the site based on which button they click.
Contact Me The purpose of this page is for potential employers to reach out to my client. The audience for this page will be potential employers. The content for this page will be a form for an employer to fill to get in contact with my client. This is a page where the user will enter data. The page will need data validations to ensure the user has entered information into required fields before submitting. The page will contain hyperlinks to the other pages, and a submit button to submit the entered information. When the user clicks a button it will navigate them to the other pages of the site based on which button they click. The submit button will also send the entered information into a database.

Dynamic Functionality on the Website

The type of interactivity I will have on this site is a contact me page that will send an email to my client through the site itself. I webpage I've seen with similar functionality is this one here: https://deaniechen.com/contact.
